# Radar API Error Codes

| Error Code | Description             |
|------------|-------------------------|
| `2000`     | Internal Error          |
| `2001`     | Input Validation Error  |
| `2002`     | Query is above max cost |
| `2003`     | Too many requests       |
| `2004`     | Not found               |
